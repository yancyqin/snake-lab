🐍 Snake Lab

A three-version snake game built as a vibe-coding learning journey for kids. Each version teaches more about programming — from a single-player classic to programmable bots.

v1 — Classic Play

Single-player snake. A big world (60×60), camera follows your snake, minimap shows where the food is. Touch, swipe, or keyboard.

▶ Play v1

v2 — Arena Local only

Multiplayer snake — up to 8 humans in a room over WebSocket. Needs a Node.js server, so it won't run on GitHub Pages. Clone the repo to play.

See on GitHub

v3 — Coder Coming

Write a JavaScript function that controls a snake. Battle other kids' bots in an arena. Includes a teacher mode for pacing in class.
